# SEEDUP  
투자 입문자를 위한 AI 기반 투자 판단 지원 서비스

---

## Overview

최근 개인 투자자는 빠르게 늘어나고 있지만,
투자 판단은 여전히 감·뉴스·분위기에 의존하는 경우가 많습니다.

문제는 정보가 부족해서가 아니라,
그 정보를 일관된 기준으로 해석하고 판단으로 연결하는 구조가 없다는 점입니다.

SEEDUP은 이 지점에서 출발했습니다.

---

## Problem

투자 입문자의 실제 상태는 다음과 같습니다.

- 매수·매도 기준이 없음  
- 정보는 많지만 판단으로 연결되지 않음  
- 투자 결과에 대한 설명이 어려움  
- 경험이 학습으로 축적되지 않음  

결국 문제는  
**“무엇을 볼지”가 아니라 “어떻게 판단할지”에 있음**  

---

## Approach

SEEDUP은 AI를 다음과 같이 정의했습니다.

- 답을 대신 내려주는 도구 ❌  
- 판단 과정을 정리해주는 도구 ⭕

서비스는 다음 흐름을 기준으로 설계되었습니다.
설문 / 가입
→ 투자 성향 파악
→ 포트폴리오 추천
→ 종목 분석
→ AI 설명
→ 사용자 판단
→ 경험 축적


각 단계는 단절된 기능이 아니라  
사용자가 판단에 도달하기 위한 하나의 흐름으로 연결됩니다.

---

## Key Features

### 1. 투자 성향 기반 추천  
사용자의 투자 목적과 리스크 성향을 반영해  
선택 가능한 종목과 포트폴리오를 제한합니다.  

→ “무엇을 볼지”를 줄이는 역할  

---

### 2. 종목 분석 (설명 중심)  
재무 데이터, 뉴스, 리포트를 통합해  
해당 종목의 현재 상태를 설명합니다.  

→ “왜 이 종목인지”를 이해하는 단계  

---

### 3. 투자 챗봇  
사용자의 질문에 대해 정답을 제공하기보다,  
현재 상황에서 고려해야 할 판단 요소를 정리합니다.  

→ “어떻게 판단할지”를 돕는 역할  

---

## Differentiation

| 기존 투자 서비스 | SEEDUP |
|----------------|--------|
| 종목 추천 중심 | 판단 과정 중심 |
| 수익률 중심 | 리스크 + 근거 중심 |
| 정보 나열 | 정보 구조화 |
| 결과 제공 | 의사결정 보조 |

---

## Structure

서비스는 사용자 맥락을 중심으로 데이터를 연결합니다.

- 사용자 정보 (성향, 보유 종목)  
- 시장 데이터 (재무, 가격)  
- 비정형 데이터 (뉴스, 리포트)  
- 대화 데이터 (챗봇 이력)  

단순 조회가 아니라  
**사용자의 현재 판단 상황을 기준으로 정보가 재구성되도록 설계했습니다.**

---

## Limitations

- 실제 매매 기능은 포함하지 않음  
- 투자 자문이 아닌 판단 보조 도구  
- 실사용 데이터 대신 가상 데이터 사용  
- 핵심 기능 중심 MVP 구현  

---

## Conclusion

SEEDUP은 투자 결과를 제공하는 서비스가 아니라,  
사용자가 스스로 판단할 수 있도록 돕는 서비스입니다.
